Hotkey EVE

Download Flattr EVE View on GitHub

EVE helps you to learn shortcuts, in order to increase your productivity with MAC OS X. Every time you execute an action using the mouse, EVE will show you the matching shortcut.

Features

Displaying
  • shortcuts for menu items
  • shortcuts for Safari GUI elements (de, en), eg. the back button
  • shortcuts for Mail GUI elements (de, en), eg. reply button
  • shortcuts for Chrome GUI elements (de, en), eg. the back button
  • shortcuts for Finder GUI elements (de, en)
  • shortcuts for iCal GUI elements (de, en)
  • notifications with Growl and OS X Notification Center
*** New in Version 1.2.0 ***
  • Mountain Lion Notification Center support
  • GUI Elements support for iCal (de, en)
  • rescaning apps and search for new shortcuts (click on the eve icon)
  • shortcut count for the actual app (click on the eve icon)
  • reset and delete the database (click on the eve icon)
  • new shortcut indexing core
  • eve is now working with sqlite3 database
  • Growl 2.0 support
  • many many improvements under the hood
*** New in Version 1.1.0 ***
  • disable notifications for shortcuts (click on a message)
  • disable all notifcations for single apps (click on a message)
  • check for updates
  • many many bugfixes and improvements
System Requirements
  • Intel
  • 64 bit
  • Mac OS X 10.6 or newer

Known Bugs

If the internal monitor is active, the support for GUI elements doesn't work on external displays.
